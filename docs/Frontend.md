# Frontend Documentation

## Framework
Next.js 14 utilizing the App Router.

## Styling
TailwindCSS is the primary utility framework, orchestrated alongside `clsx` and `tailwind-merge` for dynamic component classes.

## Components
- `src/app/(public)/components`: Public-facing presentation components (ArticleCard, Headers, Footers).
- `src/components/ui`: Shadcn-like reusable primitive components (Buttons, Inputs, Dialogs).
